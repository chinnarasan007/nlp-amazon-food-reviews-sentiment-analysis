# NLP-amazon-food-reviews-sentiment-analysis
Sentiment analysis on Amazon food reviews using Natural Language Processing techniques and machine learning models to classify reviews as positive, neutral, or negative.

## Project Overview

This project performs sentiment analysis on Amazon food product reviews using Natural Language Processing (NLP) and Machine Learning.

The model predicts whether a customer review is:

Positive  
Neutral  
Negative

## Dataset

Amazon Food Reviews Dataset

Contains more than 500,000 customer reviews including:

Product ID  
User ID  
Review Text  
Review Summary  
Rating Score

## Technologies Used

- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow

1 Data Collection  
2 Data Cleaning  
3 Text Preprocessing  
4 Feature Extraction (TF-IDF)  
5 Model Training  
6 Model Evaluation  

## Machine Learning Model

Logistic Regression

## Evaluation Metrics

Accuracy Score  
Classification Report  
Confusion Matrix

## Results

The model successfully predicts the sentiment of customer reviews and can be used to analyze customer feedback automatically.
